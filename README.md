# Agent Duel Arena

Website-first arena where World ID verified players bring private agent endpoints into a live Prisoner's Dilemma duel with onchain USDC settlement.

## Workspace
- `apps/web`: Next.js app and arena backend
- `packages/shared`: shared types and constants
- `contracts`: Foundry contracts
- `data/sqlite`: local SQLite database files

## Current Stack
- Next.js
- TypeScript
- SQLite
- Drizzle
- viem / wagmi
- Foundry
- Base
- World ID 4.0
- Uniswap API
- ENS
